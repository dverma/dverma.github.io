The Challenge...

In the TextArea element at the bottom of the page, write the JavaScript that will sort the buttons in the bottom portion of this page in ascending order based on the number of vowels in the name of each button's associated state. The button whose associated state has the fewest vowels should appear at the top.

For our purposes vowels are defined as the letters a,e,i,o, and u in both upper-case and lower-case.

When you click the "Evaluate" button, the JavaScript you've pasted into the TextArea will be executed.

The successful solution will yield a list of buttons that will appear in this order:

Texas
Florida
Alabama
California
Louisiana

Once sorted, clicking each button should still display an alert() message with the name of that associated state's capital city.

Good luck!
